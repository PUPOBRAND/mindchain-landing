# MindChain — Landing

Landing page de MindChain: juego de preguntas y respuestas sobre Bitcoin, historia del dinero y conciencia, con recompensas en sats vía Lightning.

## Stack

HTML estático autocontenido (un solo archivo `index.html`, sin build ni dependencias). Deploy en Vercel.

## Editar

Todo el contenido, estilos y JS están en `index.html`. Cualquier cambio subido a `main` se despliega automáticamente en Vercel.

## Proyecto

MindChain es parte del ecosistema de *Bitcoin sin humo*.
